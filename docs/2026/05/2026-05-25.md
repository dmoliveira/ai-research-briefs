<h2 id="executive-summary">Executive Summary</h2>

This issue is led by AI-for-research work. OpenAI's discrete-geometry milestone is the most visible item: an internal general-purpose reasoning model produced a proof that disproves the long-standing belief that square-grid constructions are essentially optimal for the planar unit-distance problem. The separate formal-proof-search paper is also important because it evaluates AI agents against open Erdős and OEIS conjectures under Lean verification, not just benchmark math problems.

The agent-systems cluster is unusually practical. DeltaBox argues that state exploration for agents is now blocked by operating-system checkpoint and rollback latency. TerminalWorld turns real terminal recordings into validated agent tasks. Spreadsheet-RL and WorkstreamBench both show that spreadsheet automation is a serious agent frontier, but current systems still fall short of professional deliverable quality.

The evaluation and safety papers are also worth attention. Agentic CLEAR, SynAE, Boiling the Frog, and LCGuard all target failure modes that show up after agents become stateful, tool-using, and multi-turn. The common thread is that model outputs are no longer the whole artifact; evaluation has to inspect traces, tools, state changes, latent channels, and synthetic-test fidelity.

    <div class="entry-meta-item"><span class="entry-meta-label">Authors</span><span>OpenAI; companion remarks by external mathematicians</span></div>

    <div class="entry-meta-item"><span class="entry-meta-label">Link</span><span><a class="entry-link" href="https://openai.com/index/model-disproves-discrete-geometry-conjecture/">https://openai.com/index/model-disproves-discrete-geometry-conjecture/</a></span></div>

    <div class="entry-section"><p><span class="entry-section-title">Summary</span>OpenAI says a general-purpose reasoning model found an infinite family of point configurations that improve the known lower-bound behavior for the unit-distance problem.</p></div>
    <div class="entry-section"><p><span class="entry-section-title">Problem investigated</span>The planar unit-distance problem asks how many pairs among n planar points can be exactly distance one apart; the prevailing belief was that square-grid-style constructions were essentially optimal.</p></div>
    <div class="entry-section"><p><span class="entry-section-title">Context</span>Mathematics is a hard but unusually checkable setting for measuring whether AI can contribute original research, because proof claims can be inspected by experts.</p></div>

    <div class="entry-section"><p><span class="entry-section-title">Conclusion</span>The result supports the view that frontier models can sometimes produce research contributions that survive expert scrutiny, not merely assist with exposition.</p></div>
    <div class="entry-section"><p><span class="entry-section-title">My take</span>This is the week’s most important read. The caveat is that it is a high-profile milestone page, not a conventional peer-reviewed paper, so the proof and companion notes matter more than the announcement itself.</p></div>

  <h3>DeltaBox: Scaling Stateful AI Agents with Millisecond-Level Sandbox Checkpoint/Rollback</h3>
  <p class="entry-summary">An OS-level sandboxing paper that enables fast checkpoint and rollback for agent search, reinforcement learning, and high-frequency state exploration.</p>
  <div class="entry-meta-grid">
    <div class="entry-meta-item"><span class="entry-meta-label">Published</span><span>2026-05-21</span></div>
    <div class="entry-meta-item"><span class="entry-meta-label">Authors</span><span>Yunpeng Dong, Jingkai He, Yuze Hou, Dong Du, Zhonghu Xu, Si Yu, Yubin Xia, Haibo Chen</span></div>
    <div class="entry-meta-item"><span class="entry-meta-label">Affiliations</span><span>Not listed on the arXiv abstract page</span></div>
    <div class="entry-meta-item"><span class="entry-meta-label">Link</span><span><a class="entry-link" href="https://arxiv.org/abs/2605.22781">https://arxiv.org/abs/2605.22781</a></span></div>
    <div class="entry-meta-item"><span class="entry-meta-label">Tags</span><span>agent sandboxes, operating systems, checkpoint rollback, test-time search, RL</span></div>
    <div class="entry-meta-item"><span class="entry-meta-label">Reading profile</span><span>16-22 min &middot; Relevance 5/5 &middot; Quality 5/5</span></div>
  </div>
  <div class="entry-section-grid">
    <div class="entry-section"><p><span class="entry-section-title">Summary</span>DeltaBox introduces change-based filesystem and process-state checkpointing so agents can branch and roll back sandbox state with millisecond latency.</p></div>
    <div class="entry-section"><p><span class="entry-section-title">Problem investigated</span>Deep search and RL for tool-using agents require frequent state forks, but conventional sandbox duplication is too slow for large fan-out.</p></div>
    <div class="entry-section"><p><span class="entry-section-title">Context</span>As agent systems move from single trajectories to search over possible actions, sandbox state becomes a runtime bottleneck.</p></div>
    <div class="entry-section"><p><span class="entry-section-title">Solutions explored</span>The paper proposes DeltaState, DeltaFS, and DeltaCR to record only state changes and roll back via layer switches and frozen process templates.</p></div>
    <div class="entry-section"><p><span class="entry-section-title">Key results</span>The abstract reports approximately 14 ms checkpoint and 5 ms rollback latency, with evaluations on SWE-bench and RL micro-benchmarks.</p></div>
    <div class="entry-section"><p><span class="entry-section-title">Conclusion</span>Agent-scale exploration needs sandbox primitives designed for branching computation, not only isolation.</p></div>
    <div class="entry-section"><p><span class="entry-section-title">My take</span>This is a practical systems paper with immediate relevance to coding agents, browser agents, and any RL loop that needs cheap trial execution.</p></div>

</article>

    <div class="entry-meta-item"><span class="entry-meta-label">Authors</span><span>Ali Hatamizadeh, Yejin Choi, Jan Kautz</span></div>
    <div class="entry-meta-item"><span class="entry-meta-label">Affiliations</span><span>NVIDIA; not fully listed on the arXiv abstract page</span></div>
    <div class="entry-meta-item"><span class="entry-meta-label">Link</span><span><a class="entry-link" href="https://arxiv.org/abs/2605.22791">https://arxiv.org/abs/2605.22791</a></span></div>
    <div class="entry-meta-item"><span class="entry-meta-label">Tags</span><span>linear attention, long context, recurrent state, efficient inference, retrieval</span></div>
    <div class="entry-meta-item"><span class="entry-meta-label">Reading profile</span><span>14-20 min &middot; Relevance 4/5 &middot; Quality 5/5</span></div>
  </div>
  <div class="entry-section-grid">
    <div class="entry-section"><p><span class="entry-section-title">Summary</span>The paper improves DeltaNet-style linear attention by giving the model separate channel-wise controls for erasing old memory and writing new content.</p></div>
    <div class="entry-section"><p><span class="entry-section-title">Problem investigated</span>Compressed recurrent attention states need to forget and update information, but tying erase and write gates can corrupt existing associations.</p></div>

    <div class="entry-section"><p><span class="entry-section-title">Conclusion</span>Better state editing can make linear-attention models more competitive for long-context tasks while keeping decoding memory bounded.</p></div>
    <div class="entry-section"><p><span class="entry-section-title">My take</span>This is less product-immediate than DeltaBox, but it is technically strong for anyone tracking alternatives to full KV-cache attention.</p></div>
